# 2025 Rule 5 Draft Analysis

## Project Overview
Comprehensive scouting analysis of MLB Rule 5 Draft-eligible pitchers and position players using advanced metrics and statistical modeling.

## Objective
Identify high-upside, cost-controlled talent eligible for the Rule 5 Draft by analyzing performance metrics, injury history, and development trajectories.

## Data Sources
- FanGraphs (Stuff+, Location+, advanced metrics)
- Baseball Savant (Statcast data)
- Baseball Reference (historical stats)
- Custom injury database

## Key Findings
- Identified 8 high-upside pitcher targets
- Identified 4 position player targets
- Found correlation between A/AA Stuff+ improvement and MLB success
- Developed risk assessment framework

- R (tidyverse, ggplot2)
- Statistical Analysis
- Data Visualization

## Business Impact
Cost-controlled talent acquisition strategy that could provide 1-2 WAR value at minimum salary (~$760k vs. $5-10M free agent market).
